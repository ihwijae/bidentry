## 변경 요약
- KEPCO 로그인 입력 안정화
  - 보이는 input만 선택하도록 필터링
  - 입력 전 클릭/포커스 강제
  - 입력 성공 여부 로그 추가
- KEPCO 로그인 속도 개선
  - 로그인 링크/모달/필드 탐색 타임아웃 및 대기 시간 단축
  - 로그인 후 네비게이션 대기 조건을 domcontentloaded로 완화
- Edge 로컬 네트워크 허용 팝업 회피
  - KEPCO에서 local-network 권한 자동 부여
  - PNA 프롬프트 비활성화를 위한 브라우저 실행 옵션 추가

## 수정된 파일
- automation-engine/src/sites/kepco.js
- automation-engine/src/web/playwright.js

## 상세 변경 내용
### 1) KEPCO 로그인 입력 안정화
- 보이는 input만 대상으로 선택해 숨겨진 필드 입력 방지
- 입력 전 click/focus로 활성화 보장
- 입력 성공 여부 로그:
  - [KEPCO] ID/PW input set (id=..., pw=...)

### 2) KEPCO 로그인 속도 개선
- 로그인 링크 탐색 타임아웃 감소
- 모달/필드 탐색 재시도 간격 감소
- 팝업 대기 시간 단축
- 로그인 후 대기 조건을 load -> domcontentloaded로 완화

### 3) 로컬 네트워크 허용 팝업 우회
- KEPCO일 때 자동으로 local-network 권한을 부여
- Edge 실행 옵션:
  - --disable-features=BlockInsecurePrivateNetworkRequests

## 동작 옵션
- 자동 local-network 권한 비활성화:
  - options.autoGrantLocalNetwork = false
- PNA 프롬프트 비활성화 비활성화:
  - options.disablePrivateNetworkPrompt = false
