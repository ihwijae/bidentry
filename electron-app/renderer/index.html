사용 가이드

1. 설정

  1. 상단 메뉴에서 설정 탭으로 이동한 뒤, 한전·국방부 URL을 입력합니다.
    각 사이트 로그인 페이지 주소를 사용하면 됩니다.
  2. 회사 관리 섹션에서 회사명을 추가합니다.
    회사명은 공인인증서에 표시된 업체명과 동일해야 인증서 매칭이 정확해집니다.
  3. 인증서 설정에서 회사별 인증서 정보(주체, 발급자, 일련번호, 매체, 경로, PIN)를 입력하고 저장합니다.
  4. 계정 정보에서 회사 선택 후 한전/국방부 ID와 비밀번호를 입력한 뒤 저장합니다.

2. 작업 실행

  1. 메인 탭의 작업 실행 카드에서 사이트(한전 또는 국방부)를 선택합니다.
  2. 공고 번호 영역에서 참가신청할 공고번호를 한 개 이상 추가합니다.
  3. 회사 선택 박스에서 방금 설정한 회사를 선택합니다.
  4. 실행 버튼을 눌러 자동 참가신청을 시작합니다.

진행 중 각 공고의 상태는 공고 진행상황 목록에서 확인할 수 있으며, 각 공고의 [복사] 버튼으로 공고번호를 클립보드에 복사할 수 있습니다.
